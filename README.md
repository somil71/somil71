## 🔨 What I've shipped

### [Needle](https://github.com/somil71/NEEDLE) — Local-first code search engine
Built entirely in Rust. Hybrid BM25 + HNSW vector search, hand-rolled from scratch (no Qdrant, no Tantivy). Tree-sitter AST chunking across 7 languages, live D3 call graph, MCP server for Claude/Cursor/Windsurf. Ships as a native desktop app (Tauri), VS Code extension, CLI, and cloud deployment.

`Rust` `Axum` `Tauri` `Tree-sitter` `HNSW` `BM25` `MCP`

### AI Auditor
